Introductie

In deze webpagina een introductie.
Dit is niet mijn eerste HTML pagina. Sterker nog ik heb er al aardig wat gemaakt.
Voor de HTML lessen die ik geef moet ik natuurlijk mijn leerlingen kunnen uitleggen hoe HTML werkt en de basis is natuurlijk altijd de structuur.

In de lessen komen o.a. de volgende opdrachten aan bod:

  1. Structuur HTML pagina
  2. Kleur achtergrond en tekst
  3. Afbeeldingen en links
  4. Extra: YouTube video toevoegen

Vervolg

Natuurlijk kijk ik uit naar het vervolg in deze cursus
Ik weet zeker dat ik nog veel ga leren. Want ook al is de basis structuur makkelijk, de echte uitdaging zit volgens mij in het toevoegen van javascript om de site interactief te maken.
Ook weet ik zeker dat ik nog veel kan leren over de opmaak met CSS.

Slot

Dit is natuurlijk nog maar een korte eerste pagina. Ik hoop snel mooiere pagina's te kunnen maken met meer typen content, zoals afbeeldingen en video's.
Tevens vind ik het belangrijk om netjes te leren programmeren, dus daar krijg ik ook graag feedback op.
